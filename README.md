# 🌈 Background Generator

## 🎨 Description
A dynamic background generator that allows users to create custom gradients with real-time previews. Users can select colors, gradient types, directions, and even apply animations.

## ✨ Features
- 🎨 Custom color selection
- 🔄 Linear and radial gradient options
- 📐 Adjustable gradient direction
- 🎭 Animated background effect
- 🎲 Random color generator
- 📱 Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript


## 📌 Usage
1. Open `index.html` in a web browser.
2. Select two colors for the gradient.
3. Choose between a linear or radial gradient.
4. Adjust the direction if using a linear gradient.
5. Click "Apply" to set the background.
6. Use "Toggle Animation" for dynamic effects.
7. Click "Random Colors" for a surprise gradient!
